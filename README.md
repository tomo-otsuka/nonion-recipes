# Nonion Recipes

Welcome to the **Nonion Recipes** collection! This repository hosts a curated selection of recipes that have been collected, created, and altered to fit a strict allium-free dietary restriction. 

It is designed to be published as a static site via GitHub Pages.

## What is Allium-Free?

An allium-free diet means consuming absolutely no ingredients from the Allium genus. This includes, but is not limited to:
- Onions (red, yellow, white, sweet)
- Garlic
- Shallots
- Leeks
- Scallions / Green Onions
- Chives
- Garlic and Onion powders

For many, this is due to dietary restrictions, allergies, or sensitivities (such as a low-FODMAP diet).

## About this Project

Cooking without onions and garlic can feel daunting, as they form the foundational flavor base (like mirepoix or sofrito) for many global cuisines. This project aims to provide delicious, flavor-packed meals that use creative substitutions—like fennel, celery, ginger, asafoetida (hing), and robust herbs—to build complex flavors without relying on alliums.
